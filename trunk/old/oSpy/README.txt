BUILDING
--------

You need a recent version of libwbxml [1] and SharpZipLib [2]. The quickest
approach is to simply download the latest oSpy binary release and extract the
following files into bin\Debug:

libwbxml2.dll
leaktrack.dll
libexpat.dll
ICSharpCode.SharpZipLib.dll

[1] http://libwbxml.aymerick.com
[2] http://www.icsharpcode.net/OpenSource/SharpZipLib/Default.aspx
